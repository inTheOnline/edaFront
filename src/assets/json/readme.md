# 路由Mate中各项的含义

1.  icon : 这是一个图标相关的配置项，表示与该页面或菜单项关联的图标。
2.  title : 这是页面的标题或菜单项的名称，通常用于显示在浏览器标签页、导航菜单或页面头部。
3.  isLink : 这个字段通常表示当前项是否是一个链接。如果为空字符串或未定义，可能意味着它不是一个链接，或者链接的处理逻辑需要在其他地方定义。
4.  isHide : 这个字段表示是否隐藏该页面或菜单项。如果为 false，表示该项是可见的。如果为 true，则该项会被隐藏，不显示在导航菜单中。
5.  isFull : 这个字段可能表示页面是否需要全屏显示。如果为 false，表示不需要全屏。如果为 true，则页面可能会以全屏模式显示。
6.  isAffix : 这个字段通常用于固定某些页面或菜单项在导航栏中。如果为 true，该项会一直显示在导航栏中，不随滚动条滚动而消失。如果为 false，则不会固定。
7.  isKeepAlive : 这个字段通常用于控制页面缓存。如果为 true，页面会在切换路由时被缓存，不会被销毁和重新创建。如果为 false，页面每次访问时都会重新加载。
